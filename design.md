# AI 简历工具设计文档

## 1. 设计概述

**品牌名称（暂定）**：ResumeCraft / 简历匠（实际使用时可根据产品名替换）  
**产品定位**：一款基于 AI 的简历优化与生成工具，帮助用户快速获得专业、精美、与目标岗位高度匹配的简历。  
**设计关键词**：Simple · Premium · Professional · Trustworthy  
**设计语言**：Modern Editorial + Soft Tech，以克制的高级感替代过度装饰的科技风。

## 2. 设计原则

1. **留白即奢华**：大面积留白，让内容呼吸，避免信息过载。
2. **色彩克制**：以白色、暖灰、深蓝为基调，蓝紫渐变仅用于 CTA 和关键状态。
3. **字体精准**：Montserrat 全站，标题字重 600-700，正文 400-500，建立清晰层级。
4. **微动效点睛**：按钮 shimmer、卡片悬浮、页面过渡，提升精致感但不干扰阅读。
5. **玻璃拟态克制使用**：仅用于导航栏和少量浮层，避免全屏玻璃带来的视觉噪音。
6. **一致性**：三套简历模板与网站界面在字体、色彩、圆角、间距上保持一致。

## 3. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#FFFFFF` | 页面大面积背景 |
| 次级背景 | `#F8FAFC` | 卡片、表单区域、上传区 |
| 深色背景 | `#0F172A` | Hero 区、Footer、科技模板侧边栏 |
| 主文字 | `#0F172A` | 标题、正文 |
| 次级文字 | `#64748B` | 辅助说明、标签 |
| 反白文字 | `#FFFFFF` | 深色背景上使用 |
| 主强调 | `#3B82F6` | 链接、按钮、选中态 |
| 次强调 | `#8B5CF6` | 渐变终点、装饰元素 |
| 成功 | `#10B981` | 完成态、下载成功 |
| 警示 | `#F59E0B` | 提示、轻警告 |
| 错误 | `#EF4444` | 错误态 |

### 渐变规范

- **Hero 背景**：`linear-gradient(135deg, #0F172A 0%, #1E293B 50%, #334155 100%)`
- **CTA 按钮**：`linear-gradient(90deg, #3B82F6 0%, #8B5CF6 100%)`
- **装饰光晕**：低透明度蓝紫模糊圆形，仅用于 Hero 和成功页

## 4. 字体系统

**字体**：Montserrat（Google Fonts）  
**备用栈**：`system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`

| 层级 | 桌面尺寸 | 字重 | 行高 | 用途 |
|------|---------|------|------|------|
| H1 | 48px | 700 | 1.15 | Hero 大标题 |
| H2 | 32px | 600 | 1.25 | 页面标题 |
| H3 | 20px | 600 | 1.35 | 卡片标题 |
| Body | 16px | 400 | 1.65 | 正文 |
| Small | 14px | 500 | 1.5 | 标签、提示 |
| Tiny | 12px | 500 | 1.4 | 辅助信息 |

## 5. 网站设计方案

### 5.1 全局导航

- **位置**：固定顶部，高度 64px
- **背景**：滚动前透明，滚动后 `rgba(255, 255, 255, 0.85)` + `backdrop-blur-md` + 底部细线
- **内容**：左侧 Logo（文字 Logo + 小图标），右侧「开始制作」按钮
- **行为**：在 Hero 区透明，离开 Hero 后变为毛玻璃白底

### 5.2 首页（Landing Page）

**结构**：
1. 全屏 Hero 区
2. 三步流程说明
3. 三风格模板缩略图展示
4. 简洁 Footer

**Hero 区**：
- 背景：深蓝渐变，叠加两个模糊光晕（蓝紫色，透明度 20-30%）
- 左侧文案：
  - 主标题：48px/700 白色，"AI 按目标岗位，优化你的简历"
  - 副标题：18px/400 `#94A3B8`，"上传简历、粘贴岗位 JD，30 秒获得三套专业设计"
  - CTA 按钮：蓝紫渐变，圆角 12px，文字 16px/600，带 hover 光晕扩散
- 右侧：三张 A4 简历模板以 3D 透视角度层叠展示，轻微悬浮动画（上下 6px 循环，时长 4s）

**三步流程**：
- 三列白色卡片，顶部带小图标（Upload、Sparkles、Download）
- 卡片 hover 时轻微上抬 `translateY(-4px)`，阴影加深
- 标题 20px，正文 14px 灰色
- 步骤文案：
  1. 上传简历：支持 PDF / Word 格式
  2. AI 优化：按目标岗位 JD 扩写、精简、改写
  3. 选择导出：一键下载 PDF 或 Word

**风格展示**：
- 三列 A4 比例缩略图
- 每列下方标注风格名称：极简 / 科技 / 精美
- 缩略图 hover 时放大 1.02 倍，过渡 200ms

### 5.3 上传页

**结构**：
1. 页面标题 + 副标题
2. 隐私声明条
3. 两栏布局：左侧上传区，右侧表单区

**页面标题**：
- 标题：32px/600 "上传简历并填写需求"
- 副标题：16px/400 `#64748B` "AI 会根据你的简历和目标岗位进行优化"

**隐私声明条**：
- 位置：标题下方
- 样式：背景 `#F0FDF4`，圆角 10px，内边距 12px 16px
- 内容：盾牌图标 + 绿色对勾 + 文字「你的简历仅在本次处理中使用，处理完成后即销毁，不会被存储或用于其他用途。」
- 文字颜色：`#166534`

**上传区**：
- 容器：虚线边框 `border-2 border-dashed border-slate-300`，圆角 16px，背景 `#F8FAFC`
- 内部：上传图标 + 主文字「点击或拖拽上传简历」+ 次文字「支持 PDF、Word，最大 10MB」
- 拖拽状态：边框变为主强调色 `#3B82F6`，背景变为 `#EFF6FF`
- 上传成功：显示文件名 + 绿色对勾图标 + 删除按钮

**需求表单**：
- 容器：白色卡片，圆角 16px，阴影 subtle
- 字段：
  1. 目标岗位 JD：大文本框，4 行，placeholder 为「粘贴目标岗位 JD，AI 会按关键词优化」
  2. 期望语气：按钮组（专业 / 沉稳 / 活泼），选中态为蓝紫渐变白字
  3. 重点突出：多选标签（业绩 / 技能 / 项目 / 领导力），选中态为蓝紫渐变白字
  4. 其他要求：可选文本框
- 提交按钮：蓝紫渐变，全宽，hover 时 shimmer 流动

### 5.4 预览页

**结构**：
1. 页面标题 "选择你喜欢的模板风格"
2. 三列 A4 比例模板预览
3. 底部确认栏

**模板卡片**：
- 每个卡片为 A4 比例（宽高比 1:1.414），圆角 8px
- 默认状态：细边框 `#E2E8F0`，轻微阴影
- Hover：边框变为主强调色，阴影加深
- 选中态：蓝色边框加粗 2px，右下角显示勾选图标
- 底部标签：风格名称（极简 / 科技 / 精美），14px/500

**交互**：
- 点击卡片选中，再次点击取消
- 点击卡片主体外区域可放大查看（灯箱效果）
- 确认按钮默认禁用，选中后启用

**底部确认栏**：
- 固定底部或位于页面底部
- 左侧：已选模板名称
- 右侧：「重新生成」文字按钮 + 「确认模板」主按钮

### 5.5 下载页

**结构**：
1. 居中展示选中模板的大图预览
2. 底部格式选择按钮组 + 重新生成入口

**设计**：
- 模板大图容器：白色背景，A4 比例，最大高度 70vh，阴影强烈
- 下载按钮：
  - 主按钮：下载 PDF
  - 次按钮：下载 Word
- 重新生成：文字按钮，位于底部
- 成功状态：按钮变为绿色对勾，提示「下载成功」

## 6. 简历模板设计方案

三套模板统一规范：
- **尺寸**：A4（210mm × 297mm）
- **边距**：上下 20mm，左右 18mm
- **字体**：Montserrat（标题）+ Inter（正文，提升可读性）
- **段落间距**：8-12mm
- **分隔线**：0.5px `#CBD5E1`，不喧宾夺主

### 6.1 模板一：极简风（Minimalist）

**关键词**：留白、克制、呼吸感、编辑风

**布局**：
- 单栏居中
- 顶部：姓名 28px/700，职位/意向一行 14px/500 灰色
- 联系信息：一行图标 + 文字，字号 11px
- 各模块标题：11px/700，全大写，字间距 1px，下方 1px 细线
- 经历条目：职位/学校 16px/600，机构/时间 14px/400 灰色，描述 14px/400
- 技能：标签形式，浅灰背景圆角

**色彩**：
- 背景：纯白 `#FFFFFF`
- 主文字：`#0F172A`
- 辅助文字：`#64748B`
- 分隔线：`#CBD5E1`
- 强调：无彩色，仅用字重和字号层级

### 6.2 模板二：科技风（Tech）

**关键词**：结构化、数据感、双栏、蓝紫点缀

**布局**：
- 双栏：左侧 30% 深蓝 sidebar，右侧 70% 主内容
- 左侧 sidebar：
  - 顶部姓名 24px/700 白色
  - 联系方式与技能，反白文字
  - 技能用进度条或点阵可视化
- 右侧主内容：
  - 模块标题 13px/700 全大写，左侧带 3px 蓝紫渐变竖条
  - 经历条目时间轴形式，左侧小圆点
  - 项目经验用卡片式条目

**色彩**：
- Sidebar：`#0F172A`
- 主背景：`#FFFFFF`
- 模块标题：`#0F172A`
- 强调：`#3B82F6` 和 `#8B5CF6` 用于小装饰

### 6.3 模板三：精美风（Elegant）

**关键词**：精致、杂志感、顶部色带、层级丰富

**布局**：
- 顶部 25mm 高色带区：深蓝渐变背景 + 白色姓名
- 下方双栏：左侧 35% 个人信息/技能，右侧 65% 经历
- 姓名：32px/700 白色，底部带细白线
- 联系信息：白色半透明小字
- 模块标题：13px/600，配合小图标
- 经历条目：职位 16px/600，机构 14px/500 蓝紫，描述 14px/400
- 技能：彩色小圆点 + 文字

**色彩**：
- 顶部色带：`#0F172A` 到 `#334155` 渐变
- 主背景：`#FFFFFF`
- 强调：`#3B82F6`
- 辅助：`#8B5CF6` 用于小标签

## 7. 动效与交互规范

### 7.1 页面过渡
- 页面切换：淡入 + 轻微上滑（`opacity 0→1, translateY 8px→0`），时长 300ms，ease-out
- 首次进入 Hero：标题整体从下方滑入，时长 600ms

### 7.2 通用 Hover
- 卡片：`-translateY(4px)`，阴影加深，过渡 200ms
- 按钮：背景渐变位置移动，产生 shimmer 效果
- 链接：颜色从 `#0F172A` 过渡到 `#3B82F6`

### 7.3 微交互
- 上传区拖拽：边框颜色变化 + 内部图标轻微放大
- 模板选中：边框加粗 + 勾选图标 scale 弹出（0 → 1.2 → 1，时长 200ms）
- 下载按钮：点击后显示 spinner，成功时变为对勾
- 表单字段 focus：边框变蓝 + 外发光

### 7.4 原则
- 动效不干扰内容阅读
- 过渡时长不超过 300ms（Hero 入场除外）
- 避免过多动画导致廉价感

## 8. 组件规范

### 8.1 按钮

| 类型 | 样式 | Hover |
|------|------|-------|
| 主按钮 | 蓝紫渐变，白色文字，圆角 12px，padding 12px 24px | 背景渐变位置移动，轻微光晕 |
| 次按钮 | 白色背景，深蓝边框 1px，深蓝文字 | 背景变浅灰 `#F8FAFC` |
| 文字按钮 | 透明背景，主色文字 | 颜色加深，下划线出现 |
| 危险按钮 | 透明背景，红色文字 | 背景浅红 |

### 8.2 卡片

- 背景：白色 `#FFFFFF`
- 圆角：16px
- 阴影默认：`0 4px 20px rgba(15, 23, 42, 0.06)`
- 阴影 hover：`0 12px 32px rgba(15, 23, 42, 0.1)`
- 过渡：200ms ease-out

### 8.3 输入框

- 边框：`1px solid #E2E8F0`
- 圆角：10px
- 背景：白色
- Padding：12px 14px
- Focus：边框 `#3B82F6`，外发光 `0 0 0 3px rgba(59, 130, 246, 0.15)`
- Placeholder 颜色：`#94A3B8`

### 8.4 标签 / 选择

- 多选标签：圆角 20px，padding 6px 14px
- 未选中：白底灰边，文字 `#64748B`
- 选中：蓝紫渐变背景，白色文字
- 过渡：150ms

### 8.5 上传区

- 默认：虚线边框，背景 `#F8FAFC`
- 拖拽中：实线边框 `#3B82F6`，背景 `#EFF6FF`
- 成功：绿色对勾 + 文件名 + 删除按钮

## 9. 响应式策略

| 断点 | 布局策略 |
|------|---------|
| ≥1024px（桌面） | 完整三栏展示，模板预览原比例，上传页双栏 |
| 768-1023px（平板） | 上传页双栏变单栏，预览页三列变两列 + 横向滚动 |
| <768px（手机） | 单列堆叠，模板预览缩小，确认按钮固定底部 |
